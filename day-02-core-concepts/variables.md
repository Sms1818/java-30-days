# ***Java Variables***

---

## **Defination**

In Java, variables are containers used to store data in memory. They define how data is stored, accessed, and manipulated.

---

## **What is a Variable?**

A variable is a named memory location used to hold data that can be changed during program execution.

---

## **Components of a Variable**

A variable in Java has three components:

* **Data Type** → Defines the type of data (e.g., int, String, float)
* **Variable Name** → Identifier used to access the variable
* **Value** → Actual data stored in the variable

---

## **Variable Declaration**

Variable declaration means defining the data type and name of the variable.

### **Syntax**

```java
dataType variableName;
```

---

## **Variable Initialization**

Assigning a value to a variable.

### **Syntax**

```java
dataType variableName = value;
```

---

## **Example**

```java
int age = 25;
String name = "Java";
double salary = 50000.50;
```

---

## **Rules for Naming Java Variables**

* Must start with a **letter (a–z, A–Z), $ or _**
* Cannot use **Java keywords** (int, class, if)
* **Case-sensitive** (age and Age are different)
* Can contain **letters, digits (0–9), $, _**
* Must not contain **spaces**
* Use **meaningful names** (studentName instead of s)
* Follow **camelCase convention** (totalMarks)

---

## **Key Characteristics**

* Java is **statically typed**
* Variables must be **declared before use**
* Supports **strong type checking**
* Improves **code readability and maintainability**
